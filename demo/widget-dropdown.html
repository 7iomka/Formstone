Basic

The most basic method is simply applying the Dropdown plugin:

$("select").dropdown();
Demo

Option Groups

Dropdown automatically detects optgroup tags, allowing you to style groups:

$("select").dropdown();
Demo

Dropdown will also automatically detect multiple select elements:

$("select").dropdown();
Demo

Default Label

The default label will be displayed on initialization only if an option does not have the selected attribute explicitly defined.

$("select").dropdown({
	label: "Select An Item"
});
Demo

Links

The options can also be links, think "jump navigation":

$("select").dropdown({
	links: true
});
Demo

External Links

The links can also open in a new window or tab:

$("select").dropdown({
	externalLinks: true
});
Demo

Cover

The options list can cover the handle, sort of Safari-style:

$("select").dropdown({
	cover: true
});
Demo

Class

Define a custom CSS class to have multiple visually distinct Dropdowns on the same page:

$("select").dropdown({
	customClass: "custom"
});
Demo

Disable

Dropdown will automatically detect disabled fields, but you can always manually trigger the disable or enable event:

$("select").dropdown("disable");

$("select").dropdown("enable");
Demo

You can also disable and enable individual options by passing that option's value:

$("select").dropdown("disable", "value");

$("select").dropdown("enable", "value");
